Project 1: Landing Page for Technology Startup

Project 1 Link

A landing page is a marketing or advertising website designed to inform the visitor about a brand, company, or product. This landing page is informative and simple. The page contains three tabs where users can click to expand the content within that tab. The first tab contains a brief introduction and history of the brand. The second tab showcases the current products with a table layout or with a slideshow, depending on the device that is accessing the website. The third tab contains a contact form.

All of the content of the website is displayed on a single page, which keeps the website simple and prevents the user from having to click on links to access the information on a separate page

This website is responsive. The content layout of the second tab changes depending on the screen width. If the user is viewing the page from a laptop or monitor, all of the current products are visible, and when the user is viewing the page from a mobile device or tablet, a slideshow displays one product at a time.

Project 2: Small Business Website for Florist

Project 2 Link

The purpose of this project was to create a website for a small business. The business, which is a florist, needs to be able to organize the different types of products they sell.

Each type of flower has its own page. The flowers are also sorted by occasion, such as Mother's Day or Valentine's Day. Having the products organized into separate pages keeps the website clean and prevents the user from being overwhelmed by images. This also drastically reduces loading time; instead of the server trying to load every single high-definition image onto one page, each page only holds a maximum of 6 high-definition images which can load much faster.

This website is also responsive. The font sizes and image width change depending on the type of device that is accessing the website. Modern websites must be developed for use with all types of devices - from mobile phones to desktop monitors - in order to be accessible for every user.
